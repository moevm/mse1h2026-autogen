Данный скрипт - класс QuestionN2, унаследованный от QuestionBase. 
Он используется для автоматической генерации различных вариантов Задачи 2 и набора тестов для нее.

Условие задачи: 
Студенту необходимо написать программу на C, которая: 
- Считывает строку символов (длиной не более maxInputSize, последним символом всегда является '\n')
- Выполняет одну из операций модификации строки:
    - удаление всех символов определённого типа: цифр, заглавных букв или строчных букв
    - циклический сдвиг всех латинских букв (a–z, A–Z) на заданное число позиций вправо или влево (шифр Цезаря)
- Выводит полученную строку

Инициализация:
- При запуске класс получает seed и maxInputSize
- На основе seed случайным образом определяется:
  - тип операции (delete или shift)
  - если delete — тип удаляемых символов (digits, upper, lower)
  - если shift — тип регистра (upper или lower), направление и величина сдвига

Методы обработки строки:

process_delete():
- Определяет регулярное выражение в зависимости от типа символов
- Удаляет соответствующие символы из строки
- Возвращает изменённую строку

shift_char():
- Если символ не относится к нужному регистру — возвращает его без изменений
- Находит индекс символа в алфавите
- Выполняет циклический сдвиг:
- Возвращает новый символ

process_shift():
- Применяет shift_char к каждому символу строки
- Формирует итоговую строку

Метод process():
- В зависимости от выбранной операции вызывает process_delete или process_shift

Метод generateGoodTest():
- Генерирует случайную длину строки
- Формирует строку из случайных цифр, строчных и заглавных букв
- Вычисляет ожидаемый результат через process
- Возвращает кортеж (programInput, expectedOutput)

Метод generateBadTest():
- Генерирует строку, которая не должна измениться:
  - при delete — строка без символов удаляемого типа
  - при shift — строка без символов нужного регистра
- Возвращает (programInput, programInput)

Генерация текста условия:
- Свойство questionText формирует HTML-текст задания
- Автоматически создаёт два примера:
  - корректный тест
  - тест, в котором строка не изменяется
- Вставляет описание конкретной операции (включая направление и величину сдвига при shift)

Предзагруженный код:
- Метод preloadedCode возвращает шаблон минимальной программы на C с пустым main

Метод test():
- Компилирует переданный код через CProgramRunner
- Генерирует 5 корректных тестов на основе seed
- Запускает программу на каждом тесте
- Сравнивает вывод с ожидаемым результатом
- Затем генерирует 5 тестов, где строка не должна измениться
- В случае несоответствия возвращает Result.Fail
- Если все тесты пройдены успешно — возвращает Result.Ok
