Данный скрипт - класс QuestionN1, унаследованный от QuestionBase. 
Он используется для автоматической генерации различных вариантов Задачи 1 и набора тестов для нее.

Условие задачи: 
Студенту необходимо написать программу на C, которая: 
- Считывает заданное количество чисел в 8, 10 или 16 системе счисления
- Может работать с 3 форматами входных данных. Числа могут быть разделены пробелом, находиться в скобках, подаваться в формате 'num1: #, num2: #, ...'
- Вычисляет сумму имеющихся чисел 
- Выводит сумму в другой системе счисления
- Обрабатывает ошибки ввода. Если хоть одно из чисел не удалось считать, необходимо вывести 'error: #''

Инициализация: 
- При запуске класс получает seed. Случайным образом определяются система счисления входных чисел и результата, формат входных данных.

BAD_NUMBERS:
- Список строк, который используется для генерации тестов с ошибочным вводом.

Методы конвертации (toInput, toOutput):
- Переводят значения int в строковый вид для нужной системы счисления.

Метод форматирования (formatInput):
- Принимает массив чисел и преобразует их в нужный формат. 

Методы для генерации тестов (generateGoodTest, generateBadTest):
- generateGoodTest() - создает корректные данные. Генерирует массив случайных чисел в одном из возможных форматов и высчитывает ожидаемую сумму.
- generateBadTest() - создает ошибочные данные. Часть чисел генерируется как обычно, остальные заменяются на значения из BAD_NUMBERS. Ожидаемый вывод системы: 'error: #'

Генерация условия (questuionText):
- Необходимо для сборки текста задачи для веб-интерфейса системы. 

Проверка решения (test):
- Комплирует код 
- Прогоняет 5 хороших тестов и 5 плохих
- В случае, если вывод программы не соответствует ожидаемому, возвращает Result.Fail
- Если все тесты пройдены, возвращает Result.Ok
