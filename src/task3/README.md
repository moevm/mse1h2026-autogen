Данный скрипт - класс QuestionN3, унаследованный от QuestionBase. 
Он используется для автоматической генерации различных вариантов Задачи 3 и набора тестов для нее.

Условие задачи: 
Студенту необходимо написать программу на C, которая: 
- Считывает число N - размер массива
- Если используется режим с битовой маской, дополнительно считывает число M - битовую маску
- Считывает N целых чисел и сохраняет их в статический массив
- Выполняет одну из операций над элементами с чётными или нечётными индексами:
    - вычисляет сумму
    - вычисляет произведение
    - вычисляет сумму элементов, у которых установлены все биты, заданные маской M

Инициализация: 
- При запуске класс получает seed. На основе seed опредяляется: тип индексов, тип операции, максимальная длина массива, если mask_sum, то генерируется еще маска M

Метод generateTest():
- Генерирует случайную длину массива
- Генерирует массив случайных чисел в зависимости от типа операции
- Формирует входную строку программы:
    - N a1 a2 ... aN
    - либо N M a1 a2 ... aN (в режиме mask_sum)
- Вычисляет ожидаемый результат
- Возвращает кортеж (programInput, expectedOutput)

Генерация текста условия:
- Свойство questionText формирует HTML-текст задания для отображения в веб-интерфейсе системы.

Предзагруженный код:
- Метод preloadedCode возвращает шаблон минимальной программы на C. 

Метод test():
- Компилирует переданный код через CProgramRunner
- Генерирует 5 тестов на основе того же seed
- Запускает программу на каждом тесте
- Сравнивает вывод с ожидаемым результатом
- В случае несоответствия возвращает Result.Fail
- Если все тесты пройдены успешно - возвращает Result.Ok
